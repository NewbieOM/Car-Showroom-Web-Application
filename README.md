# Car-Showroom-Web-Application
A responsive dynamic car showroom web application for college project  built using HTML, CSS, JavaScript, Node.js, Express.js, and MySQL.


## Features

- 🚘 Dynamic vehicle listings
- ⭐ Featured cars section
- 💬 Customer reviews
- 🔐 Login & Signup UI
- 🔑 Forgot Password UI
- 📅 Test Drive Booking
- 🛒 Car Booking
- 🔍 Compare Cars
- 📱 Responsive Design
- 🗄️ MySQL Database Integration

## Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MySQL
